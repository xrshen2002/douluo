# 写作风格指南

## 核心要求

### 段落长度要求
**重要原则：段落要充实饱满，避免一段一句话**

❌ 错误示范：
```
凌霜深吸一口气。

她看向队友们。

"准备好了吗？"她问道。

"准备好了！"众人回答。
```

✅ 正确示范：
```
凌霜深吸一口气，目光扫过身边的队友们。苏晴岚正在活动手腕，水蓝色的魂力若隐若现；林煜握紧赤刃，眼中燃烧着战意；温静书安静地调整着魂力状态；赵铭轩沉稳地站在原地，如同一座不可撼动的山岳；莫云帆推了推眼镜，星罗棋盘已经在手中缓缓展开。她能感受到每个人都已经进入了最佳状态，一个月的魔鬼训练让他们从陌生的伙伴变成了真正的战友。

"准备好了吗？"凌霜的声音平静而坚定，带着作为队长的自信。她知道，接下来的战斗将会是他们迄今为止面临的最大挑战，但她相信自己的队友，相信他们精心准备的战术，更相信这一个月来建立起的默契。

"准备好了！"五人齐声回应，声音整齐划一，透着必胜的决心。
```

### 段落构成原则

1. **场景描写段落**：融合环境、氛围、细节
   - 不要只写一个动作或一句话
   - 加入感官描写（视觉、听觉、触觉等）
   - 融入人物心理或环境氛围

2. **战斗场景段落**：动作+效果+反应
   - 攻击动作 + 魂技效果 + 对手反应 + 战场变化
   - 至少3-5句话构成一个完整的战斗段落
   - 体现攻击的视觉冲击力和战术意义

3. **对话段落**：对话+神态+动作+心理
   - 不要只有纯对话
   - 加入说话时的动作、表情、语气描写
   - 适当加入说话者的心理活动

4. **心理描写段落**：思考过程要完整
   - 展开人物的思考过程
   - 回忆、分析、判断、决策形成完整链条
   - 每个段落至少4-6句话

### 段落长度参考标准

- **最短段落**：3句话以上
- **常规段落**：5-8句话
- **重点段落**：8-12句话
- **单句段落**：仅用于强调或转折，每章不超过3处

## 武魂设计要求

### 顶级武魂设计标准

1. **独特性**
   - 每个武魂都要有鲜明的特色和视觉冲击力
   - 不能只是简单的"XX之盾"、"XX之剑"
   - 要有独特的形态、颜色、纹路、光效

2. **机制深度**
   - 不只是"防御高"、"攻击强"
   - 要有特殊机制（如千山之盾的主动镇压）
   - 要有战术价值和配合空间

3. **视觉表现**
   - 详细描写武魂的外观特征
   - 魂技释放时的视觉效果
   - 能量波动、光影效果、环境影响

### 武魂描写模板

**首次登场**：
- 外观特征（形态、大小、颜色、纹路）
- 能量气息（压迫感、元素属性）
- 魂环配置（颜色、年份、特殊之处）
- 给对手/观众的第一印象

**魂技释放**：
- 魂环亮起的视觉效果
- 魂技名称（要有气势和意境）
- 技能效果的详细描写
- 对战场/环境的影响
- 对手的反应

## 战斗场景要求

### 战斗层次结构

1. **战略层面**：整体战术布局
2. **战术层面**：技能组合、阵型变换
3. **技能层面**：每个魂技的表现和效果
4. **心理层面**：双方的心理博弈和反应

### 战斗节奏把控

**完整战斗结构**：
1. **开局试探**（10-15%）：展示双方实力和风格
2. **中期博弈**（30-40%）：战术对抗，你来我往
3. **转折点**（10-15%）：某方占据优势或出现变数
4. **高潮对决**（30-40%）：全力以赴，绝招对决
5. **结局收尾**（5-10%）：胜负揭晓，战后总结

### 战斗描写要点

**避免的问题**：
- ❌ 简单的"A攻击，B防御，A再攻击"
- ❌ 只描写动作，不描写效果
- ❌ 技能名称堆砌，缺乏画面感
- ❌ 战斗过程平铺直叙，没有起伏

**应该做到**：
- ✅ 每次攻击都有战术意图
- ✅ 技能效果有详细的视觉描写
- ✅ 体现双方的思考和应对
- ✅ 有节奏的张弛变化

## 创意战术设计

### 战术创意来源

1. **物理原理**：热胀冷缩、杠杆原理、力的分解等
2. **化学反应**：水火交融、元素克制、催化反应等
3. **兵法战术**：围魏救赵、声东击西、以逸待劳等
4. **现实运动**：篮球挡拆、足球配合、团队协作等

### 战术设计要求

1. **合理性**：符合武魂特性和等级限制
2. **创意性**：让读者眼前一亮的新奇战术
3. **层次性**：战术中有战术，明招暗招结合
4. **成长性**：随着实力提升，战术也要进化

---

## 人物刻画要求

### 主角凌霜

- **性格**：冷静、智慧、果断、有责任心
- **特点**：战术天才，善于分析，重视团队
- **成长线**：从逃亡者到领袖，从依赖战术到实力提升
- **写作要点**：多展现思考过程，少展现情绪波动

### 团队成员

每个人都要有鲜明特点：
- **苏晴岚**：活泼、速度型、蛟龙血脉觉醒线
- **林煜**：沉稳、实战派、火属性爆发
- **温静书**：温柔、治疗核心、生命系奥义
- **赵铭轩**：寡言、防御如山、玄武血脉
- **莫云帆**：智慧、战术大脑、家族传承

## 节奏控制

### 章节长度

- **常规章节**：8000-12000字
- **战斗章节**：10000-15000字
- **过渡章节**：6000-8000字

### 内容配比

**一章内容构成**：
- 30-40%：战斗/行动场景
- 20-30%：对话交流
- 15-25%：心理描写
- 10-20%：环境/背景描写
- 5-10%：过渡衔接

### 信息密度

- 避免大段纯对话
- 避免大段纯心理活动
- 保持场景切换的流畅性
- 每个场景都要有推进剧情的作用

## 常见问题清单

### 段落问题
- [ ] 是否有过多的单句段落？
- [ ] 对话段落是否加入了动作和神态？
- [ ] 战斗段落是否足够详细？

### 武魂问题
- [ ] 新武魂是否有独特性？
- [ ] 魂技效果是否描写充分？
- [ ] 视觉表现是否足够震撼？

### 战斗问题
- [ ] 战术是否有创意和深度？
- [ ] 节奏是否有张弛变化？
- [ ] 双方博弈是否充分展现？

### 情节问题
- [ ] 是否符合已有设定？
- [ ] 时间线是否准确？
- [ ] 人物等级是否合理？
- [ ] 是否为后续剧情埋下伏笔？

## 命名规范

### 命名总体原则

**核心要求**：符合玄幻小说气质，有意境和韵味，但不能过度夸张

❌ **太朴素普通的命名**：
- 武魂：铁盾、火刀、水枪
- 魂技：攻击、防御、治疗
- 战队：一队、二队、强队
- 人物：小明、小红、阿强

❌ **太夸张过度的命名**：
- 武魂：混沌至尊神帝剑、万界毁灭灭世锤
- 魂技：九天十地无敌神龙霸体、万古长空寂灭虚空斩
- 战队：宇宙最强无敌战队、诸天万界第一队
- 人物：龙傲天、叶天帝、萧破天

✅ **恰到好处的命名**：
- 武魂：千山之盾、焚天赤刃、苍澜水蛟、星罗棋盘
- 魂技：雷霆万钧、寒冰囚笼、蛟龙缠绕、玄武壁障
- 战队：初霜小队、雷霆队、铁壁队、影刃队
- 人物：凌霜、苏晴岚、林煜、赵铭轩

### 武魂命名规范

**命名结构**：【属性/特征】+【核心意象】+【类型】

**正面示例**：
- 千山之盾（数量+意象+类型）
- 焚天赤刃（程度+天地+颜色+类型）
- 苍澜水蛟（颜色+意象+生物）
- 星罗棋盘（天象+排列+物品）
- 生命之树（属性+意象）

**命名要点**：
1. 使用古雅词汇：澜、焚、罗、玄、苍等
2. 融入天地意象：天、地、山、河、星、月等
3. 体现属性特征：冰、火、雷、风、光、暗等
4. 不超过五个字
5. 朗朗上口，有韵律感

**属性词库参考**：
- 冰系：寒、霜、冰、雪、冽、冻、凝
- 火系：焚、炎、烈、赤、焰、灼、燎
- 雷系：雷、霆、电、闪、轰、震、劫
- 水系：澜、潮、波、涛、渊、泉、溟
- 土系：山、岳、磐、岩、峰、石、壤
- 风系：风、岚、飙、翼、羽、翔、疾
- 光系：曜、辉、耀、烁、煌、明、昭
- 暗系：影、暗、幽、冥、夜、墟、渊

**意象词库参考**：
- 天地：天、地、苍穹、大地、九天、九幽
- 日月：日、月、星、辰、曜、宿
- 山川：山、河、岳、川、峰、涧、壑
- 神兽：龙、凤、麟、龟、虎、豹、鹰、蛟
- 兵器：剑、刀、枪、戟、锤、斧、盾、弓
- 神物：树、花、玉、珠、镜、鼎、炉、塔

### 魂技命名规范

**命名结构**：【效果/意象】+【核心动作】

**正面示例**：
- 雷霆万钧（属性+程度+量词）
- 寒冰囚笼（属性+效果）
- 蛟龙缠绕（形态+动作）
- 玄武壁障·真武（武魂+效果+进阶形态）
- 双雷破空斩（数量+属性+效果+动作）

**命名要点**：
1. 三到五个字为佳
2. 体现技能效果
3. 符合武魂特性
4. 有画面感和冲击力
5. 进阶技能可加·后缀

**动作词库**：
- 攻击：斩、击、轰、破、裂、崩、灭、绝
- 防御：壁、盾、护、守、罡、障、阵、域
- 控制：困、锁、缚、封、镇、压、禁、囚
- 增益：祝、赐、佑、护、灵、光、韵、泽
- 移动：闪、遁、瞬、步、影、形、游、梭

**效果词库**：
- 范围：天罗、地网、漫天、遍地、四方、八方
- 程度：万钧、千重、百炼、无双、至尊、极致
- 形态：龙、凤、虎、蛇、鹰、豹、熊、狼
- 意境：破空、裂天、崩地、震岳、穿云、贯日

### 战队命名规范

**命名结构**：【核心特征】+队

**正面示例**：
- 初霜小队（主角武魂核心+小队）
- 雷霆队（属性特征）
- 铁壁队（战术风格）
- 影刃队（战术+武器）
- 狂风队（属性+气势）

**命名要点**：
1. 两到三个字+队
2. 体现队伍特色
3. 简洁有力
4. 易于记忆

**风格词库**：
- 元素：冰、火、雷、风、光、暗、土、水
- 气势：狂、烈、霸、威、凌、煞、锋、刃
- 防御：铁、钢、磐、岩、盾、壁、堡、垒
- 速度：疾、风、影、电、光、闪、瞬、雷
- 战术：谋、智、阵、策、奇、诡、幻、变

### 人物命名规范

**命名结构**：【姓氏】+【名字（2字）】

**正面示例**：
- 凌霜（姓+单字，简洁有气质）
- 苏晴岚（姓+双字，温婉优雅）
- 林煜（姓+单字，火属性相关）
- 赵铭轩（姓+双字，稳重大气）
- 莫云帆（姓+双字，飘逸洒脱）
- 温静书（姓+双字，文雅恬静）

**命名要点**：
1. 姓氏选择常见或古雅姓氏
2. 名字与性格、武魂相呼应
3. 避免太俗气或太怪异
4. 注意音韵和谐
5. 体现人物气质

**常用姓氏库**：
- 单字姓：林、苏、赵、莫、温、云、叶、陆、沈、江
- 双字姓：司马、欧阳、上官、慕容、东方、南宫、西门、北冥

**名字用字参考**：
- 冰系角色：霜、雪、冰、寒、冽、凝、晶
- 火系角色：煜、焰、炎、烈、焚、灼、耀
- 水系角色：岚、澜、涛、渊、泽、沁、溪
- 雷系角色：震、霆、鸣、轰、闪、霄、电
- 文雅型：书、雅、静、婉、韵、诗、琴、棋
- 刚强型：铭、轩、峰、岳、磐、岩、钢、铁
- 灵动型：岚、云、帆、羽、翼、影、梦、幻

### 融合技命名规范

**命名结构**：【参与者特征】+【效果】+【动作】

**正面示例**：
- 双雷天降（数量+属性+效果+动作）
- 双雷破空斩（数量+属性+效果+动作）
- 铁壁锁天阵（特征+效果+类型）

**命名要点**：
1. 体现参与者数量或特征
2. 突出融合后的威力
3. 四到六个字
4. 气势要强
5. 朗朗上口

### 阵法/领域命名规范

**命名结构**：【特征】+【效果】+阵/域/界

**正面示例**：
- 铁壁锁天阵（特征+效果+阵）
- 迷阵棋局（效果+意象）
- 雷云领域（属性+意象+领域）

**命名要点**：
1. 体现阵法核心功能
2. 三到五个字
3. 有神秘感
4. 符合武魂特性

### 命名检查清单

每次创建新名字时，检查以下几点：
- [ ] 是否太朴素普通？
- [ ] 是否太夸张过度？
- [ ] 是否符合玄幻气质？
- [ ] 是否朗朗上口？
- [ ] 是否与角色/武魂/技能特性相符？
- [ ] 是否有画面感和意境？
- [ ] 字数是否合适（不宜过长）？
- [ ] 是否容易记忆？
