# 写作风格指南

## 核心要求

### 段落长度要求
**重要原则：段落要充实饱满，避免一段一句话**

❌ 错误示范：
```
凌霜深吸一口气。

她看向队友们。

"准备好了吗？"她问道。

"准备好了！"众人回答。
```

✅ 正确示范：
```
凌霜深吸一口气，目光扫过身边的队友们。苏晴岚正在活动手腕，水蓝色的魂力若隐若现；林煜握紧赤刃，眼中燃烧着战意；温静书安静地调整着魂力状态；赵铭轩沉稳地站在原地，如同一座不可撼动的山岳；莫云帆推了推眼镜，星罗棋盘已经在手中缓缓展开。她能感受到每个人都已经进入了最佳状态，一个月的魔鬼训练让他们从陌生的伙伴变成了真正的战友。

"准备好了吗？"凌霜的声音平静而坚定，带着作为队长的自信。她知道，接下来的战斗将会是他们迄今为止面临的最大挑战，但她相信自己的队友，相信他们精心准备的战术，更相信这一个月来建立起的默契。

"准备好了！"五人齐声回应，声音整齐划一，透着必胜的决心。
```

### 段落构成原则

1. **场景描写段落**：融合环境、氛围、细节
   - 不要只写一个动作或一句话
   - 加入感官描写（视觉、听觉、触觉等）
   - 融入人物心理或环境氛围

2. **战斗场景段落**：动作+效果+反应
   - 攻击动作 + 魂技效果 + 对手反应 + 战场变化
   - 至少3-5句话构成一个完整的战斗段落
   - 体现攻击的视觉冲击力和战术意义

3. **对话段落**：对话+神态+动作+心理
   - 不要只有纯对话
   - 加入说话时的动作、表情、语气描写
   - 适当加入说话者的心理活动

4. **心理描写段落**：思考过程要完整
   - 展开人物的思考过程
   - 回忆、分析、判断、决策形成完整链条
   - 每个段落至少4-6句话

### 段落长度参考标准

- **最短段落**：3句话以上
- **常规段落**：5-8句话
- **重点段落**：8-12句话
- **单句段落**：仅用于强调或转折，每章不超过3处

## 武魂设计要求

### 顶级武魂设计标准

1. **独特性**
   - 每个武魂都要有鲜明的特色和视觉冲击力
   - 不能只是简单的"XX之盾"、"XX之剑"
   - 要有独特的形态、颜色、纹路、光效

2. **机制深度**
   - 不只是"防御高"、"攻击强"
   - 要有特殊机制（如千山之盾的主动镇压）
   - 要有战术价值和配合空间

3. **视觉表现**
   - 详细描写武魂的外观特征
   - 魂技释放时的视觉效果
   - 能量波动、光影效果、环境影响

### 武魂描写模板

**首次登场**：
- 外观特征（形态、大小、颜色、纹路）
- 能量气息（压迫感、元素属性）
- 魂环配置（颜色、年份、特殊之处）
- 给对手/观众的第一印象

**魂技释放**：
- 魂环亮起的视觉效果
- 魂技名称（要有气势和意境）
- 技能效果的详细描写
- 对战场/环境的影响
- 对手的反应

## 战斗场景要求

### 战斗层次结构

1. **战略层面**：整体战术布局
2. **战术层面**：技能组合、阵型变换
3. **技能层面**：每个魂技的表现和效果
4. **心理层面**：双方的心理博弈和反应

### 战斗节奏把控

**完整战斗结构**：
1. **开局试探**（10-15%）：展示双方实力和风格
2. **中期博弈**（30-40%）：战术对抗，你来我往
3. **转折点**（10-15%）：某方占据优势或出现变数
4. **高潮对决**（30-40%）：全力以赴，绝招对决
5. **结局收尾**（5-10%）：胜负揭晓，战后总结

### 战斗描写要点

**避免的问题**：
- ❌ 简单的"A攻击，B防御，A再攻击"
- ❌ 只描写动作，不描写效果
- ❌ 技能名称堆砌，缺乏画面感
- ❌ 战斗过程平铺直叙，没有起伏

**应该做到**：
- ✅ 每次攻击都有战术意图
- ✅ 技能效果有详细的视觉描写
- ✅ 体现双方的思考和应对
- ✅ 有节奏的张弛变化

## 创意战术设计

### 战术创意来源

1. **物理原理**：热胀冷缩、杠杆原理、力的分解等
2. **化学反应**：水火交融、元素克制、催化反应等
3. **兵法战术**：围魏救赵、声东击西、以逸待劳等
4. **现实运动**：篮球挡拆、足球配合、团队协作等

### 战术设计要求

1. **合理性**：符合武魂特性和等级限制
2. **创意性**：让读者眼前一亮的新奇战术
3. **层次性**：战术中有战术，明招暗招结合
4. **成长性**：随着实力提升，战术也要进化

## 人物刻画要求

### 主角凌霜

- **性格**：冷静、智慧、果断、有责任心
- **特点**：战术天才，善于分析，重视团队
- **成长线**：从逃亡者到领袖，从依赖战术到实力提升
- **写作要点**：多展现思考过程，少展现情绪波动

### 团队成员

每个人都要有鲜明特点：
- **苏晴岚**：活泼、速度型、蛟龙血脉觉醒线
- **林煜**：沉稳、实战派、火属性爆发
- **温静书**：温柔、治疗核心、生命系奥义
- **赵铭轩**：寡言、防御如山、玄武血脉
- **莫云帆**：智慧、战术大脑、家族传承

## 节奏控制

### 章节长度

- **常规章节**：8000-12000字
- **战斗章节**：10000-15000字
- **过渡章节**：6000-8000字

### 内容配比

**一章内容构成**：
- 30-40%：战斗/行动场景
- 20-30%：对话交流
- 15-25%：心理描写
- 10-20%：环境/背景描写
- 5-10%：过渡衔接

### 信息密度

- 避免大段纯对话
- 避免大段纯心理活动
- 保持场景切换的流畅性
- 每个场景都要有推进剧情的作用

## 常见问题清单

### 段落问题
- [ ] 是否有过多的单句段落？
- [ ] 对话段落是否加入了动作和神态？
- [ ] 战斗段落是否足够详细？

### 武魂问题
- [ ] 新武魂是否有独特性？
- [ ] 魂技效果是否描写充分？
- [ ] 视觉表现是否足够震撼？

### 战斗问题
- [ ] 战术是否有创意和深度？
- [ ] 节奏是否有张弛变化？
- [ ] 双方博弈是否充分展现？

### 情节问题
- [ ] 是否符合已有设定？
- [ ] 时间线是否准确？
- [ ] 人物等级是否合理？
- [ ] 是否为后续剧情埋下伏笔？
